Nesta etapa, comecei revisando os principais comandos da linguagem Python que seriam necessários para trabalhar com dados. Aprendi a utilizar variáveis para armazenar informações, listas para guardar vários valores, dicionários para relacionar códigos e nomes, estruturas de decisão como if e else, e estruturas de repetição como for.
Também estudei a criação de funções. Uma função é um bloco de código criado para executar uma tarefa específica e pode ser reutilizado várias vezes. Por exemplo, uma função pode receber um número e devolver esse número formatado, ou receber um código e devolver um nome correspondente. O uso de funções ajuda a deixar o código mais organizado, evita repetições e facilita a manutenção.
Na primeira exploração de dados, aprendi a observar quais informações existem em uma tabela, identificar nomes de colunas, verificar valores vazios e entender o significado de cada variável. Esse processo é importante antes de qualquer análise, pois permite descobrir se os dados estão completos, se existem erros e quais informações podem ser usadas posteriormente.
Os principais aprendizados desta etapa foram: organização do código, uso de funções, interpretação de tabelas, estruturas básicas do Python e importância de conhecer os dados antes de realizar cálculos.
