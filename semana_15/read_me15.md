Depois de estudar os fundamentos de Python, a exploração de dados, a biblioteca Pandas, os gráficos com Matplotlib e as interfaces com Tkinter, foi possível juntar todos esses conhecimentos em um único sistema.
Primeiramente, foram utilizados os conceitos básicos de Python para organizar o código em funções. Funções como carregar dados, filtrar informações, calcular estatísticas, desenhar gráficos, atualizar a tela e exportar relatórios foram criadas para deixar o programa mais claro e dividido em tarefas.
Em seguida, a biblioteca Pandas foi utilizada para ler os arquivos de dados, limpar valores inválidos, selecionar informações importantes e juntar tabelas diferentes. Os cálculos estatísticos permitiram apresentar totais, porcentagens, média e mediana. Os gráficos foram usados para transformar esses resultados em informações visuais, facilitando comparações e análises.
Por último, a interface Tkinter foi utilizada para criar uma janela interativa. Nela, o usuário pode escolher filtros, comparar informações, visualizar gráficos, consultar rankings e exportar relatórios em CSV e TXT. Dessa forma, o sistema reúne análise de dados, estatística, visualização e interação com o usuário.
O principal aprendizado foi entender que o desenvolvimento de um sistema segue etapas. Primeiro é necessário conhecer os dados e aprender os comandos básicos. Depois, é preciso tratar e organizar as informações. Em seguida, os dados podem ser analisados com cálculos e gráficos. Por fim, uma interface pode ser criada para apresentar os resultados de maneira mais prática e acessível.
