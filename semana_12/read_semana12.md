Nesta etapa, aprofundei meus conhecimentos sobre análise de dados, cálculos estatísticos, criação de gráficos e organização de resultados. Depois de aprender a carregar, limpar e filtrar os dados, foi necessário transformar essas informações em resultados que pudessem ser entendidos com mais facilidade.
Um dos principais aprendizados foi o cálculo de estatísticas. Aprendi a calcular o total de registros, porcentagens, média e mediana. O total mostra a quantidade de informações analisadas. A porcentagem permite comparar uma parte dos dados com o total. A média é calculada somando todos os valores e dividindo pela quantidade de registros. Já a mediana é o valor que fica no meio quando os dados são colocados em ordem crescente.
Também aprendi a organizar esses cálculos dentro de funções. Em vez de repetir os mesmos comandos várias vezes, é possível criar uma função que recebe uma tabela de dados e devolve os resultados principais. Por exemplo, uma função pode calcular a quantidade total de pessoas, a porcentagem de uma resposta, a média de idade e a mediana. Isso deixa o código mais organizado, facilita a reutilização e torna futuras alterações mais simples.
Outra parte importante foi o estudo de gráficos com a biblioteca Matplotlib. Os gráficos ajudam a visualizar os dados de maneira mais clara. O gráfico de barras é utilizado para comparar valores entre grupos ou categorias. Por exemplo, ele pode mostrar diferenças de porcentagens entre regiões ou grupos de pessoas. Já o histograma é usado para mostrar como os valores estão distribuídos. Ele separa os dados em faixas e mostra a quantidade de registros em cada faixa, sendo útil para analisar idade, renda ou outros valores numéricos.
Nesta etapa, também aprendi sobre algoritmos de ordenação. Um algoritmo de ordenação é um conjunto de passos usados para colocar valores em uma ordem específica, como do maior para o menor ou do menor para o maior. A ordenação é importante quando se deseja criar rankings, listas de maiores valores, menores valores ou organizar informações para facilitar a análise.
O algoritmo estudado foi o Bubble Sort, também conhecido como ordenação por bolha. Ele funciona comparando dois valores vizinhos de uma lista. Se eles estiverem na ordem errada, o algoritmo troca suas posições. Esse processo é repetido várias vezes até que todos os valores estejam organizados.
Por exemplo, se uma lista possui valores de porcentagem, o Bubble Sort compara o primeiro com o segundo. Se o segundo valor for maior e a intenção for ordenar do maior para o menor, os dois valores trocam de posição. Depois, ele compara o próximo par de valores e continua fazendo as trocas. Ao final de cada passagem pela lista, um dos maiores valores vai ficando mais próximo do início da lista. Por isso, o nome “bolha” representa a ideia de valores maiores subindo gradualmente para a posição correta.
Apesar de não ser o algoritmo mais rápido para listas muito grandes, o Bubble Sort é importante para o aprendizado porque mostra de forma clara como uma ordenação acontece internamente. Ele permite entender o uso de estruturas de repetição, comparações com `if`, troca de posições e organização de listas.
Os principais aprendizados desta etapa foram cálculos estatísticos, criação de funções para reutilizar análises, elaboração de gráficos com Matplotlib, comparação de grupos, interpretação visual dos dados e compreensão de algoritmos de ordenação, especialmente o Bubble Sort.
