Nesta etapa, avancei para conteúdos intermediários da linguagem C. O principal aprendizado foi o uso de funções. Uma função é um bloco de código criado para realizar uma tarefa específica. Ela ajuda a dividir um programa grande em partes menores, evitando repetição de código e facilitando a organização.
Em C, uma função pode receber valores chamados parâmetros e pode devolver um resultado usando o comando return. Por exemplo, é possível criar uma função para calcular a soma de dois números, outra para verificar se uma pessoa foi aprovada e outra para mostrar um menu de opções. A função main() também é uma função, sendo a primeira a ser executada pelo programa.
Também aprendi sobre vetores, que são estruturas usadas para armazenar vários valores do mesmo tipo. Um vetor pode guardar, por exemplo, várias notas, idades ou números. Cada posição do vetor possui um índice, e em C a primeira posição sempre começa no índice zero. Para acessar ou alterar os valores de um vetor, são utilizados colchetes, como notas[0].
Além dos vetores, estudei matrizes. Uma matriz funciona como uma tabela com linhas e colunas. Ela pode ser usada para guardar informações como notas de vários alunos em diferentes matérias, valores de uma tabela ou posições de um jogo. Para trabalhar com vetores e matrizes, normalmente são usados laços de repetição, principalmente o for.
Outro conteúdo importante foi a noção de ponteiros e memória. Ponteiros são variáveis que armazenam endereços de memória. Eles são muito utilizados em C porque permitem trabalhar de forma mais direta com os dados armazenados pelo computador. Mesmo sendo um assunto mais avançado, entender a ideia de memória ajuda a compreender melhor como as variáveis funcionam internamente.
Também aprendi a utilizar bibliotecas, como stdio.h, que permite usar comandos de entrada e saída, e math.h, que possui funções matemáticas. As bibliotecas adicionam recursos ao programa e evitam que seja necessário criar tudo do zero.
Os principais aprendizados desta etapa foram criação de funções, uso de parâmetros e retorno, vetores, matrizes, laços de repetição aplicados a estruturas de dados, noções de memória e ponteiros, além do uso de bibliotecas. Esses conteúdos ajudam a criar programas maiores, mais organizados e com mais possibilidades.
