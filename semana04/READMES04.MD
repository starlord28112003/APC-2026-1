Nas primeiras etapas do curso, aprendi os conceitos básicos de programação no Game Lab. Comecei entendendo como funciona o desenho na tela, utilizando formas simples como círculos, retângulos e linhas. Isso me ajudou a compreender como os elementos visuais são construídos e posicionados no espaço do jogo.

Em seguida, aprendi sobre variáveis, que servem para armazenar informações importantes como pontuação e valores que podem ser alterados durante o jogo. Esse conceito foi essencial para criar jogos mais dinâmicos.

Também aprendi sobre números aleatórios, que permitem gerar variações dentro do jogo, como posições diferentes para objetos aparecerem, tornando o jogo menos previsível.

Depois disso, comecei a trabalhar com sprites, que são os personagens e objetos do jogo. Aprendi a criá-los, adicionar imagens e entender como eles funcionam dentro do jogo.

Em seguida, estudei as propriedades dos sprites, como posição, tamanho e velocidade, que permitem controlar o comportamento dos elementos do jogo.

Por fim, aprendi a exibir texto na tela, o que é importante para mostrar informações como pontuação e mensagens para o jogador.
