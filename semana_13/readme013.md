Nesta etapa, comecei a estudar a biblioteca Tkinter, utilizada para criar interfaces gráficas em Python. Uma interface gráfica permite que o usuário utilize um programa por meio de uma janela, com botões, textos, campos de seleção e gráficos, sem precisar digitar comandos diretamente no terminal.
O primeiro passo foi aprender a importar os componentes necessários. O módulo principal é importado com import tkinter as tk. Também podem ser utilizados módulos adicionais, como ttk, que oferece componentes com aparência mais moderna, filedialog, usado para abrir janelas de salvar arquivos, e messagebox, usado para mostrar mensagens de erro, aviso ou sucesso.
Depois, aprendi a criar a janela principal usando tk.Tk(). Essa janela é a base de toda a interface. Nela, é possível definir título, tamanho mínimo, cor de fundo e outras configurações. Comandos como title(), geometry(), minsize() e configure() ajudam a personalizar a aparência inicial do programa.
Também estudei os principais componentes visuais da interface. O Label é usado para mostrar textos, títulos e informações. O Button cria botões que executam ações. O Frame serve para dividir a janela em áreas organizadas. Já o LabelFrame cria uma área com borda e título, sendo útil para separar filtros, estatísticas e outras seções.
Outro conceito importante foi a organização dos elementos na tela. O Tkinter possui métodos como pack(), grid() e place(). O método pack() organiza os componentes em sequência, geralmente de cima para baixo ou de um lado para o outro. O método grid() organiza os componentes em linhas e colunas, sendo muito útil para formulários e áreas com filtros. O método place() permite definir posições exatas, mas é menos utilizado em interfaces que precisam se adaptar ao tamanho da janela.
Essa etapa ensinou como criar a estrutura visual de um programa, dividir a tela em áreas e organizar elementos de forma clara. Os principais aprendizados foram criação de janelas, configuração visual, uso de componentes básicos e métodos de posicionamento.
