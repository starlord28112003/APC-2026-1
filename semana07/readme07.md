Nesta etapa, iniciei o estudo da linguagem C, entendendo que ela é uma linguagem de programação muito importante para a computação. A linguagem C é utilizada para desenvolver sistemas operacionais, programas de baixo nível, aplicativos, jogos, sistemas embarcados e diversas outras aplicações. Ela é considerada uma linguagem próxima do funcionamento do computador, pois permite trabalhar de forma mais direta com memória, variáveis e comandos básicos.
Os primeiros conteúdos estudados foram a estrutura de um programa em C, o uso de bibliotecas, a função principal main(), variáveis, tipos de dados e comandos de entrada e saída. A função main() é o ponto onde o programa começa a ser executado. Para mostrar informações na tela, é utilizado o comando printf(). Para receber informações digitadas pelo usuário, pode ser utilizado o comando scanf().
Também aprendi sobre tipos de dados, como int para números inteiros, float para números decimais, char para caracteres e double para números decimais com maior precisão. Esses tipos são importantes porque indicam ao computador qual tipo de informação será armazenada em cada variável.
Além da linguagem C, estudei conceitos básicos de Assembly e Little Man Computer. Assembly é uma linguagem de baixo nível, mais próxima das instruções que o processador entende. Em vez de comandos mais completos, como em C, o Assembly utiliza instruções simples para carregar valores, somar, subtrair, guardar dados e fazer desvios no programa.
O Little Man Computer, também chamado de LMC, é uma ferramenta didática usada para representar de forma simples o funcionamento de um computador. Ele possui uma memória formada por caixas numeradas, uma calculadora para realizar operações, uma caixa de entrada para receber dados e uma caixa de saída para mostrar resultados. Cada instrução é representada por um código numérico.
No LMC, é possível criar programas simples, como ler dois números, somá-los e mostrar o resultado. Para isso, o programa recebe o primeiro número, guarda na memória, recebe o segundo número, soma os dois valores e envia o resultado para a saída. Esse processo ajuda a entender como o computador executa instruções passo a passo.
Os principais aprendizados desta etapa foram estrutura inicial da linguagem C, variáveis, tipos de dados, entrada e saída, funcionamento básico de Assembly e compreensão da lógica de um computador por meio do Little Man Computer.
